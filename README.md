# kkserver
HTTP server written in c#. 
# Features
## Version 1.1:
* Add methods: HEAD, OPTIONS
* Add new log-view
* Add plugins function
* Sample plugins: reverseProxy, IPFilter
## Version 1.0:
* Error pages
* Default pages
* Log making
* GUI interface
* Transfer encodings: ranges, chunks (for php interpreter and others) and normal.
